## 📖 نظرة عامة
**YosriPal** هي إضافة (Tweak) مصممة خصيصاً لبيئة الجيلبريك، تهدف إلى تحسين وتوسيع وظائف تطبيق فيسبوك الأساسي. تمنحك الأداة تحكماً كاملاً وتجربة خالية من الإعلانات مع ميزات حصرية تلبي احتياجات المستخدم المتقدم، مع التركيز التام على استقرار الأداء وتوفير استهلاك البطارية.

---

## ✨ المميزات الأساسية (Features)

*   📥 **تحميل الفيديوهات:** دعم كامل وسريع لتحميل كافة أنواع الفيديوهات بضغطة زر.
*   🚫 **حظر الإعلانات (AdBlock):** إخفاء كافة أنواع الإعلانات والمقترحات المزعجة من واجهة التطبيق.
*   👻 **الوضع الخفي (Ghost Mode):** مشاهدة القصص (Stories) دون علم أصحابها.
*   👍 **تأكيد الإعجاب:** إضافة زر تأكيد لمنع الإعجابات العشوائية أو الخاطئة.
*   🛑 **التحكم بالتشغيل:** إيقاف التنقل التلقائي (Auto-scroll) للفيديوهات.
*   ⚡ **واجهة تحكم ذكية:** واجهة خفيفة، سريعة الاستجابة، مدمجة بسلاسة مع إعدادات التطبيق.
*   🧹 **تنظيف الذاكرة:** ميزة تنظيف الكاش (Cache) لتوفير مساحة التخزين في جهازك.
*   🛠️ **اختصارات سريعة:** أزرار مخصصة للتحميل السريع عبر الضغط المطول.

## 📱 التوافق (Compatibility)
*   **نظام التشغيل:** iOS 14.0 إلى iOS 16.5
*   **المعالجات المدعومة:** Arm64 / Arm64e
*   **بيئات الجيلبريك:** متوافقة مع أحدث بيئات الجيلبريك (Rootful / Rootless).
